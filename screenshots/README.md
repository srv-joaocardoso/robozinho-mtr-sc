# Pasta de Screenshots

Esta pasta contém capturas de tela geradas automaticamente pelo sistema durante a execução. As capturas de tela são úteis para auditoria e depuração de erros.

## Estrutura

- **Erro:** Capturas de tela geradas quando ocorre um erro no sistema.
- **Processamento:** Capturas de tela relacionadas ao processamento de MTRs.

## Observações

- Certifique-se de não versionar capturas de tela que contenham informações sensíveis ou confidenciais.
- Esta pasta está incluída no `.gitignore` para evitar que screenshots sejam enviadas ao repositório.
